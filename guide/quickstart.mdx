## 运行环境

使用本系统需要如下运行环境：


<AccordionGroup>
  <Accordion icon="spider-web" title="Web服务">
    Nginx 或 Apache 等支持 GCI 服务的 Http 软件。
  </Accordion>
  <Accordion icon="php" title="PHP">
    - 需要最低 8.1 版本的 PHP 软件。
    - 需要 composer 包管理器。
  </Accordion>
  <Accordion icon="database" title="数据库">
    - 需要最低 5.7 版本的 Mysql 服务端软件。
    - 需要最新版本的 Redis 服务端软件。
  </Accordion>
</AccordionGroup>

## 部署系统

<AccordionGroup>

<Accordion icon="download" title="下载源代码">
  下载源代码到环境中的站点目录，您可以选择以下2种下载方式：
  1. 下载压缩包
  访问[github](https://github.com/duxweb/duxcms/releases)，下载我们最新的发布版。
  2. 在站点目录中使用 Composer 命令下载：
  ```sh
  composer create-project duxweb/duxcms ./
  ```
</Accordion>
<Accordion icon="gear" title="设置运行目录">
  通过集成环境中的运行目录或通过配置 Nginx 将`运行目录`指向到站点的 `public` 目录中。
</Accordion>
<Accordion icon="expand" title="检查扩展">
  通过集成环境中的 PHP 扩展配置保证开启 `zip` `redis` `fileinfo` `curl` `bcmath` 扩展。
</Accordion>
<Accordion icon="bolt" title="安装依赖">
  通过集成环境中自带或手动安装的 Composer 运行如下命令：

  ```sh
  composer install
  ```

  如果下载失败请设置为阿里云镜像：

  ```sh
  composer config -g repo.packagist composer https://mirrors.aliyun.com/composer/
  ```
</Accordion>
<Accordion icon="wand-magic" title="安装向导">
  访问站点域名将自动跳转至安装向导，根据向导继续完成系统安装：
  ```
  http://域名/
  ```
</Accordion>
<Accordion icon="lock" title="访问后台">
  安装完毕后则可通过以下域名来访问后台：
  ```
  http://域名/manage
  默认账号：admin
  默认密码：admin
  ```
</Accordion>

</AccordionGroup>
