## 使用说明

该功能为了防止误操作，不通过后台进行备份和恢复，请在当前目录下执行 `dux` 命令进行备份和恢复，备份和恢复只支持 `mysql` 数据库，并且仅支持默认配置的数据库，备份的数据库文件在 `data/backup` 目录内。

## 备份数据库

请在命令行中执行以下命令备份数据库。

```sh
php dux db:backup
```

## 恢复数据库

请在命令行中执行以下命令恢复数据库，恢复的数据库为备份的最新文件。

```sh
php dux db:restore
```
